# INVEARN - Blockchain, AI, DevOps & Cloud Solutions

Modern, interactive website for INVEARN built with React, TypeScript, and HTML5 Canvas animations.

## 🚀 Features

- **Interactive Canvas Animations**: 2000+ animated particles with mouse repulsion effects
- **Geometric Animations**: Rotating hexagons and pulsing circles
- **Dynamic Particle Connections**: Real-time particle linking system
- **Smooth Navigation**: Seamless scrolling between sections
- **Performance Optimized**: 60 FPS animations with native Canvas API
- **Responsive Design**: Mobile-friendly and adaptive layouts
- **Modern Color Scheme**: Blue, navy blue, and turquoise theme

## 🛠️ Tech Stack

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **Tailwind CSS v4** - Styling
- **HTML5 Canvas** - Interactive animations
- **Lucide React** - Icon library
- **Motion (Framer Motion)** - Additional animations

## 📦 Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🎯 Canvas Animation Features

- **2000+ Particles**: Smooth particle system with physics
- **Mouse Interaction**: Particles repel from cursor movement
- **Dynamic Connections**: Lines connecting nearby particles
- **Geometric Shapes**: Rotating hexagons and pulsing circles
- **Performance Monitor**: Real-time FPS tracking
- **Customizable Controls**: Toggle effects and adjust particle count
